# Credit Card Fraud Detection

## Project Overview

This project detects **fraudulent credit card transactions** using machine learning models on highly **imbalanced datasets**.

It focuses on **precision, recall**, and **anomaly detection methods** to effectively identify fraudulent transactions while minimizing false positives.

## Dataset

The dataset used is available from Kaggle: [Credit Card Fraud Detection Dataset](https://www.kaggle.com/mlg-ulb/creditcardfraud)

* Contains transactions made by European cardholders
* 284,807 transactions, 492 fraud cases (highly imbalanced)

> Dataset is **not included** on GitHub due to size and privacy concerns.

## Objectives

* Handle imbalanced datasets
* Detect fraudulent transactions
* Optimize precision and recall
* Apply anomaly detection methods

## Machine Learning Methods

* Logistic Regression
* Random Forest
* XGBoost
* Isolation Forest (for anomaly detection)
* SMOTE (oversampling for imbalance)

## Technologies Used

## Project Workflow

1. Load Dataset
2. Data Exploration
3. Data Preprocessing
4. Handling Imbalanced Classes
5. Model Training
6. Model Evaluation (Precision, Recall, F1-Score)
7. Visualization of Fraud Patterns

## Example Output
